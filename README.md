# Personal Portfolio Website

A minimalist, dark-themed personal portfolio website reflecting an indie/alternative aesthetic. Built with a focus on clean code and user experience, this site serves as the central hub for my academic journey in Computer Engineering & AI, technical projects, and technical writing.

## 🌐 Live Demo
You can visit the live version of the website here: **https://iambyk.github.io/**

## ✨ Key Features
* **Bilingual Architecture (EN/TR):** Fully integrated language toggle to cater to both local and global connections.
* **Minimalist Dark UI:** Designed with a clean, dark-themed aesthetic that prioritizes readability and smooth navigation.
* **Integrated Sections:**
    * **About & Education:** Details about my background, including my specialization in Artificial Intelligence.
    * **Experience:** Timeline of my professional growth, internships, and research goals.
    * **Projects Showcase:** Direct links and summaries of my software engineering and machine learning repositories.
    * **Writings:** A dedicated section highlighting my published articles on Medium and LinkedIn.
* **Downloadable CV:** A one-click resume download for recruiters and academic researchers.
* **Contact Form:** Streamlined communication channel.

## 🛠️ Technologies & Tools
* **Markup & Styling:** HTML5, CSS3, **Tailwind CSS** (for rapid, utility-first styling and responsive design).
* **Deployment:** Hosted and continuously deployed via **GitHub Pages**.
